01. 進入 專案目錄
D:\>cd D:\Developers\SourceCodes\GitHub\MyDocker\spring-boot-test

02. 建立 Dockerfile
D:\>Dockerfile

03. 建置 docker image (kevinhung/spring-boot-test)
D:\>docker build -t kevinhung/spring-boot-test .
D:\>docker buildx build --platform linux/amd64,linux/arm64,linux/arm/v7,linux/arm/v6 -t kevinhungd/spring-boot-test .


04. 查看 docker image 是否建立成功
D:\>docker images

05. 執行 image 將外部 port 80 => 內部 port 8080
D:\>docker run -p 80:8080 kevinhung/spring-boot-test

06. 登入 Docker Hub
D:\>docker login -u kevinhung -p hxxxxxxxx

07. 設定 image 的 Tag
D:\>docker tag kevinhung/spring-boot-test kevinhung/spring-boot-test

08. 將 image push 到 Docker Gub
D:\>docker push kevinhung/spring-boot-test
